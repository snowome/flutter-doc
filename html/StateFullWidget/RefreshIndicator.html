RefreshIndicator组件

Material Design下拉刷新指示器，包装一个可滚动widget。

RefreshIndicator 一定要配置列表 ListView 使用。


RefreshIndicator(
    child: ListView(        // 配合列表使用
        children: [],
    ),
    onRefresh: _handleRefresh
)

Future<Null> _handleRefresh() async {
    await Future.delayed(Duration(microseconds: 200));      // js 的 setTimeout
    return null;
}

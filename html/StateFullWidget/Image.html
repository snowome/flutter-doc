Image组件

图片

使用key 从AssetBundle获得的图像


Image(
    width: 180,
    height: 160,
    image: AssetImage('images/avatar.png'),
),
Image.asset(
    'images/avatar.png',
    width: 26,
    height: 26,
)

网络图片


Image.network(
    'http://www.devio.org/img/avatar.png',
    width: 100,
    height: 100,
),

从本地文件中获取图片，绝对路径


import 'dart:io';
Image.file(
    File(file.path),                        // file是一个变量
    File('/sdcard/Download/Stack.png'),     // 完整路径，手机的sd卡下的图片，sdcard在不同手机上的前缀不一样。
    width: 120,
    height: 90,
    fit: BoxFit.fill,
),

网络图片加载占位符


import 'package:transparent_image/transparent_image.dart';          // transparent_image: ^2.0.0
Stack(
    children: [
        // 圆形的进度指示器
        Center(
            child: CircularProgressIndicator(),
        ),
        // 淡入的从内存中加载图片
        Center(
            child: FadeInImage.memoryNetwork(
                placeholder: kTransparentImage,  // transparent_image 插件提供
                image: 'http://www.devio.org/img/avatar.png',
            ),
        ),
        // 首先加载一张本地图片，等网络图片加载显示出来后，隐藏本地图片
        Center(
            child: FadeInImage.assetNetwork(
                placeholder: 'images/loading.png',
                image: 'http://www.devio.org/img/avatar.png',
            ),
        ),
    ],
),

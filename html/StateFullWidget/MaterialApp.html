MaterialApp组件

MaterialApp是Flutter开发中最常用的符合Material Design（原质化设计）设计理念的入口Widget，它里面封装了应用程序实现Material Design所需要的一些基本组件。

通常放在页面里的根节点。


@override
Widget build(BuildContext context) {
    return MaterialApp(
        title: 'StateFullWidget',           // 标题
        theme: ThemeData(                   // 主题
            primarySwatch: Colors.blue,
        ),
        home: Scaffold(                     // 整个页面

        ),
    );
}

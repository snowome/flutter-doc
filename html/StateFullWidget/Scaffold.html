Scaffold组件

Material Design布局结构的基本实现。

此类提供了用于显示 顶部导航栏AppBar、底部导航栏BottomNavigationBar、侧边栏的API。借用Scaffold可以轻松的实现一个Flutter的页面。


@override
Widget build(BuildContext context) {
    return MaterialApp(
        theme: ThemeData(
            primarySwatch: Colors.blue,
        ),
        home: Scaffold(
            appBar: AppBar(),                                 // 顶部导航栏
            bottomNavigationBar: BottomNavigationBar(),       // 底部导航栏
            floatingActionButton: FloatingActionButton(       // 右下角悬浮按钮
                onPressed: null,
                child: Text('点我'),
            ),
            body: Text('list'),
        ),
    );
}

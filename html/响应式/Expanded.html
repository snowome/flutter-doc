Expanded组件

Expanded常用于Row, Column, Flex的子元素；

Expanded可以动态调整Expanded的子组件沿主轴的尺寸，比如设置尺寸比例、填充剩余空间。


Column(
    children: [
        Text('列表'),
        Expanded(   // 填充剩余空间，这里 Container 填满了垂直方向的 Column 除去 Text('列表') 的空间。
            child: Container(
                alignment: Alignment.center,
                decoration: BoxDecoration(color: Colors.red),
                child: Text('拉升填满高度'),
            ),
        ),
    ],
),

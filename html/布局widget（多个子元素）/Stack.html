Stack组件

可以允许其子widget简单的堆叠在一起。

一个 Positioned 的widget必须是 Stack 的子组件，且 Positioned 组件的作用是用来控制widget在 Stack 的位置。


Stack(
    children: [
        Image.network(
            'http://www.devio.org/img/avatar.png',
            width: 100,
            height: 100,
        ),
        Positioned(
            left: 0,
            bottom: 0,
            child: Image.network(
                'http://www.devio.org/img/avatar.png',
                width: 50,
                height: 56,
            ),
        ),
    ],
),

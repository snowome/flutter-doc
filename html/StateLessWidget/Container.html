Container组件

Container是Flutter里很常用的一个组件，类似于html中的div。


Container(
    margin: EdgeInsets.all(10),
    alignment: Alignment.center,
    height: 100,
    decoration: new BoxDecoration(
        color: Colors.grey,
        border: new Border.all(width: 2.0, color: Colors.red),
        borderRadius: new BorderRadius.all(new Radius.circular(20.0)),
        image: new DecorationImage(
            image: new AssetImage('graphyics/face.jpg'),
        ),
    ),
    child: Text('类似于html中的div'),
),

属性

  • ● width 宽度，包含padding值。double，比如100

  • ● height 高度，包含padding值。double，比如100

  • ● padding 内边距

    例：四个方向都设定内边距为10

    padding: EdgeInsets.all(10),

  • ● margin 外边距

    例：四个方向都设定外边距为10

    margin: EdgeInsets.all(10),

  • ● alignment child 对齐的属性，可以设置居中、居左、居右、居上、居下等等

    居中： alignment: Alignment.center,

  • ● decoration 设置装饰器，比如边框、圆角、背景图片等。

    请见 常用函数->BoxDecoration

  • ● color 背景颜色。

    例：设置背景颜色。color: Colors.red,

  • 注意：不能给 Container 同时设置 decoration 和 color 属性。如果要设置，给decoration设置color就可以。

  • ● transform 变换效果，比如翻转、旋转、变形等

    例：旋转

    transform: Matrix4.rotationZ(0.1),

  • ● child 子组件。

    例：child: Text('类似于html中的div'),

InkWell

InkWell组件在用户点击时出现“水波纹”效果


InkWell(
    onTap: (){},
    splashColor: Colors.red,                // 水波纹的颜色
    highlightColor: Colors.blue,            // 高亮颜色是按住时显示的颜色
    child: Text('这是InkWell点击效果'),
)

说明： 给字体添加边距和圆角边框，扩大“水波纹”效果，如下。发现“水波纹”超出的了圆角边框，如何解决这个问题呢？ 请使用：Ink 


InkWell(
    onTap: (){},
    child: Container(
        padding: EdgeInsets.symmetric(horizontal: 20,vertical: 8),
        decoration: BoxDecoration(
            border:Border.all(color: Colors.blue),
            borderRadius: BorderRadius.all(Radius.circular(30))
        ),
        child: Text('这是InkWell点击效果'),
    ),
)

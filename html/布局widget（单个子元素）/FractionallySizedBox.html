FractionallySizedBox组件

将水平、垂直方向撑满整个屏幕，或者说百分比。约束他里边的布局能够撑多大。


Column(
    children: [
        FractionallySizedBox(
            widthFactor: 1,
            child: Container(
                decoration: BoxDecoration(color: Colors.red),
                child: Text('宽度撑满'),
            ),
        ),
    ],
),

属性

  • ● widthFactor 水平撑满百分比，1代表100%

  • ● heightFactor 垂直撑满百分比，1代表100%<

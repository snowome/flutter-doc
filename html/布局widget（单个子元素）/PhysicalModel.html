PhysicalModel组件

可以将布局显示成不同的形状


PhysicalModel(
        color: Colors.transparent,
        borderRadius: BorderRadius.all(Radius.circular(20)),
        clipBehavior: Clip.antiAlias,
        child: PageView(
            children: [
                _item('page1', Colors.purpleAccent),
                _item('page2', Colors.orange),
                _item('page3', Colors.blue),
            ],
        ),
),

_item(String title, Color color) {
    return Container(
        alignment: Alignment.center,
        decoration: BoxDecoration(color: color),
        child: Text(
            title,
            style: TextStyle(fontSize: 20, color: Colors.white),
        ),
    );
}

属性

  • ● shape 设置裁剪形状，默认矩形

    矩形： shape: BoxShape.rectangle,

    圆形： shape: BoxShape.circle,

  • ● color 背景颜色，Color类型

    Colors.transparent,

  • ● borderRadius 圆角，BorderRadius类型

    四个角都是10个数值的圆角 BorderRadius.all(Radius.circular(10));

  • ● clipBehavior 设置裁剪行为

    不裁剪： Clip.none,

    有锯齿： Clip.hardEdge,

    抗锯齿： Clip.antiAlias,（默认）

    抗锯齿+保存图层： Clip.antiAliasWithSaveLayer

  • ● child 子组件，Widget类型

数据类型

一、数字：num 是 int、double 的父类


    num num1 = -1.0;
    int int1 = 17;
    double double1 = 23.47;

    print('绝对值 ：${ num1.abs() }');
    print('转换成 int ：${ num1.toInt() }');
    print('转换成 double ：${ num1.toDouble() }');

二、字符串


    String str1 = '请看控制台输出的字符串';
    print( str1.substring(1, 3) );     // 结果：看控
    print( str1.indexOf('控制台') );   // 结果：2

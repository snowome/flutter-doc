数据类型

一、dynamic、var、Object三者的区别：

  • ● dynamic 是动态数据类型，例：dynamic x = 'hsl'; x = 123; 在编译时不管你的数据类型，在运行的时候根据值确定数据类型。不要直接使用，有隐患。
  • ● var 不是数据类型，是一个关键字。意思是我不关心它的类型是什么，系统自动推断。数据类型一旦定义，不能修改。
  • ● Object 是 dart对象的基类，Object a1 = 'hello'; 你可以调用a1的方法，但如果你调用a1.foo();时，类型检查会报错，因为不存在这个方法。
  • ● print(变量.runtimeType);打印运行时的数据类型

二、数字：num 是 int、double 的父类


    num num1 = -1.0;
    int int1 = 17;
    double double1 = 23.47;

    print('绝对值 ：${ num1.abs() }');
    print('转换成 int ：${ num1.toInt() }');
    print('转换成 double ：${ num1.toDouble() }');

六、Map类型


    Map student1 = {'aaa': '小明', 'bbb': '小红'};
    student1['ccc'] = '张三';

    student1.forEach((key, value) {
      print('$key，$value');
    });

    Map student2 = student1.map((key, value) {
      return MapEntry(value, key); // MapEntry是key和value交换位置
    });
    print(student2); // 结果：{小明: aaa, 小红: bbb, 张三: ccc}

    for (var key in student1.keys) {
      // student1.values是获得值
      print('$key, ${student1[key]}');
    }

    student1.remove('ccc'); // 删掉了ccc
    print(student1);

    print(student1.containsKey('aaa')); // 是否包含 key 是 aaa 的值
    print(student1.containsValue('小明')); // 是否包含 value 是 小明 的值

BoxDecoration

装饰器：BoxDecoration 继承自 Decoration。


BoxDecoration(
    // 背景颜色
    color: Colors.amber,
    // 背景图片
    image: DecorationImage(
        image: NetworkImage('http://www.devio.org/img/avatar.png')
    ),
    // 边框
    border: Border.all(width: 5.0, color: Colors.red),
    // 边框圆角
    borderRadius: BorderRadius.circular(36),
    // 阴影
    boxShadow: [
        BoxShadow(
            color: Colors.greenAccent.shade200,
            offset: const Offset(5, 5),
            blurRadius: 10.0,
            spreadRadius: 20.0,
        ),
        BoxShadow(
            color: Colors.yellow,
            offset: const Offset(0, 0),
            blurRadius: 0,
            spreadRadius: 0,
        ),
    ],
    // 渐变
    gradient: LinearGradient(
        begin: Alignment.topRight,
        end: Alignment.bottomLeft,
        colors: [Colors.blue, Colors.red],
    ),
    // 背景混合模式
    backgroundBlendMode: BlendMode.exclusion,
    // 形状
    shape: BoxShape.rectangle,
),

属性

  • ● color： 背景颜色，类型->Color。

  • ● image： 背景图片，类型->DecorationImage

  • ● border： 边框，类型->Border

  • ● borderRadius： 边框圆角，类型->BorderRadius

  • ● boxShadow： 阴影，类型->List<BoxShadow>

  • ● gradient： 渐变，类型->LinearGradient->Gradient

  • ● backgroundBlendMode： 背景混合模式，类型->BlendMode

  • ● shape： 形状，类型->BoxShape

EdgeInsets

设定四个方向中每个方向上的偏移量

通常用于盒子的四个边的偏移，例如，内边距、外边距等。


Container(
    height: 100,

    margin: EdgeInsets.only(top: 10),               // 上
    margin: EdgeInsets.only(left: 10),              // 左
    margin: EdgeInsets.only(bottom: 10),            // 下
    margin: EdgeInsets.only(right: 10),             // 右

    margin: EdgeInsets.all(10),                     // 上下左右四个边

    margin: EdgeInsets.fromLTRB(5, 10, 10, 0),      // 分别设定四个边

),

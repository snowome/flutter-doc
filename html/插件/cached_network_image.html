cached_network_image

从网络上加载图片，插件会将图片缓存到本地。下次再次显示图片的时候，从本地加载图片。

cached_network_image: ^3.2.1


import 'package:cached_network_image/cached_network_image.dart';

使用占位符


Center(
    child: CachedNetworkImage(
        imageUrl: "http://www.devio.org/img/avatar.png",
        placeholder: (context, url) => CircularProgressIndicator(),
        errorWidget: (context, url, error) => Icon(Icons.error),
    ),
),

使用进度指示器


Center(
    child: CachedNetworkImage(
        imageUrl: "http://www.devio.org/img/avatar.png",
        progressIndicatorBuilder: (context, url, downloadProgress) => CircularProgressIndicator(value: downloadProgress.progress),
        errorWidget: (context, url, error) => Icon(Icons.error),
    ),
),

当您想要同时拥有占位符功能并想要让 imageprovider 在另一个小部件中使用时，您可以提供一个 imageBuilder：


Center(
    child: CachedNetworkImage(
        imageUrl: "http://www.devio.org/img/avatar.png",
        imageBuilder: (context, imageProvider) => Container(
            decoration: BoxDecoration(
                image: DecorationImage(image: imageProvider, fit: BoxFit.cover, colorFilter: ColorFilter.mode(Colors.red, BlendMode.colorBurn)),
            ),
        ),
        placeholder: (context, url) => CircularProgressIndicator(),
        errorWidget: (context, url, error) => Icon(Icons.error),
    ),
),
